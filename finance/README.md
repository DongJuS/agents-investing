# 💰 finance — 경영/회계 기록

프로젝트 운영과 관련된 **비용·구독·결제·예산** 기록 폴더입니다.
돈과 관련된 모든 문서는 이 폴더에 남깁니다.

## 목적

- 구독/결제 내역 원문 보존 (청구서·활성화 스크린샷·요금표)
- 월별 운영비 추적
- 무료 티어 초과 시점 조기 감지
- 해지·갱신 타이밍 기록

## 파일 명명 규칙

- 구독 서비스: `{서비스명}.md` (예: `cloudflare-r2.md`)
- 월별 청구 요약: `billing-YYYY-MM.md` (추후)
- 예산 계획: `budget-YYYY.md` (추후)

## 원칙

- **금액은 반드시 통화 단위와 함께 기록** ($0/month, ₩11,000/월)
- **날짜는 절대 날짜로** (2026-04-16) — "오늘", "지난주" 금지
- **무료 티어 한도와 초과 요율을 분리 기록**
- 민감한 결제 정보(카드번호, CVC)는 **절대 기록하지 않음**
- API 키·시크릿도 금지 (`.env`에만 존재)
